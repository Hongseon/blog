# 블로그에 오신 것을 환영합니다!

이것은 Astro와 Tailwind CSS로 만든 정적 블로그의 첫 번째 포스트입니다.

## 주요 기능

이 블로그는 다음과 같은 기능을 제공합니다:

- **마크다운 지원**: 마크다운으로 글을 작성할 수 있습니다
- **태그 및 카테고리**: 포스트를 태그와 카테고리로 분류할 수 있습니다
- **월별 관리**: 포스트는 월별로 자동으로 그룹화됩니다
- **반응형 디자인**: 모바일, 태블릿, 데스크톱에서 모두 잘 보입니다

## 마크다운 예제

### 코드 블록

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
